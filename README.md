# Trip Booking System (Javascript)

A small JavaScript mini-project that demonstrates **defensive prgramming**, **stateful business logic** and **clear error handling**

## Features
- Books a trip by 'id'
- Prevents double booking
- Returns helpful message for missing trips
- Validates input to avoid runtime crashes

## Tech
- JavaScript (Node.js)
- No frameworks - logic-first

## Project Structure
- 'src/trips.js' - data (array of trip objects)
- 'src/bookings.js' - booking logic (business rules)
- 'src/index.js' - demo runner (calls the functions and prints results)

## How to Run
```bash
npm install
npm start
